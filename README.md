# ChooseWho

A simple, mobile-first web application to determine the "First Player" for board games, card games, or any situation requiring a random selection.

## 🚀 Features

- **Multi-Touch Support**: Tracks up to 6 fingers simultaneously.
- **Premium Aesthetics**: Ultra-dark theme with glowing neon indicators and high-quality animations.
- **Haptic-Inspired Visuals**: Intuitive feedback for touch registration and selection.
- **Zero Dependencies**: Built with pure Vanilla JavaScript, CSS, and HTML.

## 🕹️ How to Use

1. Open the app on a touch-enabled or desktop device.
2. Place two or more fingers on the screen. For desktop, click to add.
3. Keep fingers still for **3 seconds**. For desktop, press Start.
4. The app will randomly select one person to be the "First Player".
5. Remove all fingers to reset. For desktop, click Reset or click to add to start over.

## 🛠️ Development

This project was built using:
- **HTML5** (Touch Events API)
- **CSS3** (Keyframe animations, Flexbox)
- **Vanilla JavaScript** (ES6+)
